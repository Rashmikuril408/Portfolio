IrisInsight

Predicts Iris flower species using machine learning. Built with Python & Scikit-learn.

Live Link

Project Overview

üîß How It Was Built ‚Äì Methods, Workflow This project was developed as part of my personal learning to understand the basics of supervised machine learning and model evaluation techniques. The Iris dataset, one of the most well-known datasets in pattern recognition, was used to train and test the model.
üß† Steps Followed:
1. Data Collection: The dataset was sourced from Scikit-learn‚Äôs built-in datasets. It contains 150 records of iris flowers, categorized into 3 species: Iris-setosa, Iris-versicolor, and Iris-virginica.
2. Data Preprocessing: The data was loaded using Pandas, and null/missing values were checked. Data distributions were visualized using Seaborn pair plots to understand how features like sepal/petal length and width relate to flower species.
3. Data Visualization: Matplotlib and Seaborn were used to create heatmaps, scatter plots, and histograms for exploratory data analysis (EDA). This helped identify which features were most influential in classification.
4. Model Training: The dataset was split into training (80%) and testing (20%) sets using train_test_split. Multiple models were applied and compared:
K-Nearest Neighbors (KNN)
Logistic Regression
Decision Tree Classifier
Support Vector Machines (SVM)
5. Model Evaluation: Each model was evaluated using:
Accuracy Score
Confusion Matrix
Classification Report (Precision, Recall, F1-Score)
The best-performing model achieved over 95% accuracy on the test set.
6. Deployment (Optional based on your setup): The notebook was converted into a Python script or used Streamlit/Flask to deploy a simple web interface. The model can be accessed through a web browser for live predictions.


üß©How It Works & Real-World Applications

The trained model takes four numerical inputs ‚Äî sepal length, sepal width, petal length, and petal width ‚Äî and predicts the species of the iris flower in real-time.
~ A user provides the input values through a web interface or Python script.
~ These inputs are passed to the trained machine learning model.
~ The model compares the values with patterns it has learned from the training data.
~ It returns the predicted class (Setosa, Versicolor, or Virginica) with high accuracy.
