Project Overview

üîß How It Was Built ‚Äì Methods, Workflow, and Tools Used
This project was created to assist students and institutions in predicting the chances of a student's university admission based on academic and profile parameters using supervised machine learning techniques.
üß† Steps Followed:
1. Data Collection: The dataset included parameters such as GRE Score, TOEFL Score, CGPA, SOP, LOR, University Rating, and Research Experience, sourced from public datasets on Kaggle.
2. Data Preprocessing: Missing values were handled and numeric features were normalized using StandardScaler. Categorical data (Research: Yes/No) was encoded using LabelEncoder.
3. Exploratory Data Analysis (EDA): Feature distributions and relationships were visualized using Matplotlib and Seaborn. A correlation matrix was analyzed to identify key predictors of admission chances.
4. Model Building: The following regression models were implemented and compared:
‚Äì Linear Regression
‚Äì Ridge Regression
‚Äì Lasso Regression
‚Äì Random Forest Regressor
‚Äì XGBoost Regressor
Hyperparameter tuning was done using GridSearchCV.
5. Model Evaluation: Models were evaluated using:
‚Äì R-squared Score
‚Äì Mean Squared Error (MSE)
‚Äì Root Mean Squared Error (RMSE)
6. Deployment: A user-friendly web interface was built using Streamlit, where users can input academic details and receive admission probability instantly.
